# 🚀 Quick Start Guide

## Your Zero-Based Budgeting App is Ready!

### What Changed?

Your expense tracker now follows **zero-based budgeting** principles. Every rupee must have a job!

## 📱 New Tab: "Allocate"

This is where the magic happens. At the start of each month:

1. **See Your Income**: Total income for the month
2. **See Unallocated Money**: Money not yet assigned
3. **Distribute to Envelopes**: Give every rupee a job
4. **Goal**: Make "Unallocated" = ₹0

### Key Features:
- 💵 Income summary card
- 📊 Allocated amount
- 💸 Unallocated amount (should be ₹0!)
- 🔄 Transfer between envelopes button
- 📦 Envelope cards showing:
  - Allocated amount (editable)
  - Spent amount
  - **REMAINING amount** (most important!)
  - Progress bar

## 📊 Updated Dashboard

Now focuses on what you have LEFT, not what you spent:

### Before:
```
Food: ₹2,800 / ₹5,000 (56%)
```

### After:
```
Food: ₹2,200 left
₹2,800 of ₹5,000
```

### Why?
- Instant decision making
- No mental math
- Clear spending limits

## 💳 Smart Transaction Modal

When adding an expense, you'll see warnings:

### ⚠️ No Budget Set
```
No budget set for Food this month.
Set a budget in the Allocate tab.
```

### ⚠️ Low Balance
```
Only ₹500 left in Food
```

### 🚫 Over Budget
```
⚠️ This will exceed your budget by ₹300!
```

You can still save the transaction, but you're warned!

## 📦 Renamed "Budget" → "Envelopes"

The Budget tab is now called "Envelopes" for clarity. This is where you:
- Create new envelopes
- Delete envelopes
- Assign categories (Need/Want/Saving)
- View yearly budget overview

```

```

## 🔄 Common Actions

### Transfer Money Between Envelopes:
1. Go to Allocate tab
2. Click "🔄 Transfer Between Envelopes"
3. Select From envelope
4. Select To envelope
5. Enter amount
6. Click Transfer

### Check Remaining Balance:
1. Open Dashboard
2. Scroll to Budget Overview
3. See "₹X left" for each envelope
4. Click envelope to see transactions

### Add Expense with Warning:
1. Click + button
2. Enter amount
3. Select envelope
4. See warning if over budget
5. Decide: Save anyway or adjust

### View Transactions by Envelope:
1. Dashboard → Click envelope card
2. Or Transactions tab → Filter by envelope
3. See all expenses for that envelope

## 🆘 Quick Troubleshooting

**Q: Unallocated shows negative number?**
A: You allocated more than your income. Reduce some allocations.

**Q: Can I still spend if over budget?**
A: Yes, but you'll see a warning. Consider transferring money first.

**Q: What if I forgot to allocate?**
A: Go to Allocate tab anytime and distribute your income.

**Q: Can I change allocations mid-month?**
A: Yes! Adjust anytime in the Allocate tab.

**Q: What happens to unused money?**
A: It stays in the envelope and rolls over to next month.

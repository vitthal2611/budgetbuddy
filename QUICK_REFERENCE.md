# 🚀 Quick Reference Guide

## One-Page Overview of All Features

### 📦 Main Navigation
```
Envelopes → Dashboard → Reports → Transactions → Settings
```

---

## 🎯 Core Features

### Envelopes Tab
- View all envelopes grouped by category
- See remaining balance for each
- Click "Fill Envelopes" to allocate budget
- Click envelope to view transactions
- FAB button (+) to add expense

### Dashboard Tab
- Summary cards (income, expenses, savings)
- Today's expenses
- Payment method balances
- Budget progress by category

### Reports Tab (NEW)
- **Overview**: Summary + pie chart + top expenses
- **Trends**: 6-month line chart + insights
- **Breakdown**: Bar chart by envelope

### Transactions Tab
- View all transactions
- Filter by date, type, envelope
- Edit/delete transactions
- Search functionality

### Settings Tab
- **Envelopes**: Create/edit/delete envelopes
- **Payment Methods**: Manage payment methods
- **Recurring**: Set up recurring transactions
- **Templates**: Save/load budget templates
- **Preferences**: Customize app behavior

---

## ⚡ Quick Actions

### Add Transaction
1. Click FAB (+) button
2. Select type (income/expense/transfer)
3. Fill details
4. Save

### Fill Envelopes
1. Go to Envelopes tab
2. Click "💰 Fill Envelopes"
3. Enter amounts for each envelope
4. Click "Done"

### Create Recurring Transaction
1. Settings → Recurring
2. Click "+ Add Recurring"
3. Set frequency and details
4. Save

### Save Budget Template
1. Fill your envelopes
2. Settings → Templates
3. Click "+ Save Current Budget"
4. Name your template

### Apply Rollover
1. Automatic modal appears at month start
2. Review unused budget
3. Click "Apply Rollover" or "Skip"

## ⚙️ Preferences

### Rollover Mode
- **Automatic**: Auto-add unused to next month
- **Manual**: Ask me each month
- **None**: Start fresh each month

### Block Overspending
- **ON**: Cannot add expense when envelope empty
- **OFF**: Show warnings only

### Budget Start Day
- Set which day your budget cycle starts (1-31)

### Notifications
- Low balance warnings
- Over budget warnings
- Monthly reminders

## 🐛 Troubleshooting

### Issue: Unallocated money showing
**Solution**: Go to Envelopes → Fill Envelopes → Distribute all income

### Issue: Can't add expense (strict mode)
**Solution**: Transfer money from another envelope first

### Issue: Rollover not appearing
**Solution**: Check Preferences → Rollover Mode is not "None"

### Issue: Charts not showing data
**Solution**: Ensure you have transactions for selected month

### Issue: Sync not working
**Solution**: Check internet connection, refresh page

## 📞 Quick Help

### Common Questions

**Q: What's the difference between Dashboard and Reports?**
A: Dashboard shows current month overview. Reports show trends and detailed analysis.

**Q: Can I change envelope categories?**
A: Yes! Settings → Envelopes → Select category dropdown

**Q: How do I handle irregular expenses?**
A: Create envelope, allocate monthly (annual cost ÷ 12), build up balance

**Q: What if I overspend an envelope?**
A: Transfer from another envelope or enable strict mode to prevent

**Q: Can I use on multiple devices?**
A: Yes! Sign in with same account, data syncs automatically

## 🚀 Getting Started (New Users)

### 5-Minute Setup
1. **Sign up** with email/password
2. **Create envelopes** (Settings → Envelopes)
   - Add 5-10 envelopes
   - Assign categories
3. **Add payment methods** (Settings → Payment Methods)
   - Bank accounts, cards, cash
4. **Add income** (FAB → Income)
   - Log your income
5. **Fill envelopes** (Envelopes → Fill Envelopes)
   - Distribute all income
   - Make unallocated = ₹0

**You're ready to start budgeting!** 🎉
